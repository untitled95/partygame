# 🍻 小姐牌 - 在线抽牌喝酒游戏

一个支持多人在线联机的抽牌喝酒游戏，基于经典的"小姐牌"规则。

## ✨ 功能特点

- 📱 支持手机端访问，移动端适配
- 👥 多人实时在线（最多10人）
- 🔄 自动轮流抽牌
- 🃏 完整的手牌系统
- 🎯 丰富的卡牌规则

## 🎴 卡牌规则

| 牌 | 名称 | 规则 |
|---|------|------|
| A | 点杀🎯 | 指定任意一人喝酒 |
| 2 | 小姐牌👸 | 陪酒并说"大爷您喝好"（保留手中） |
| 3 | 逛三园🦁 | 轮流说动物园/水果园/蔬菜园里有什么 |
| 4 | 找人PK⚔️ | 选人PK，输的喝 |
| 5 | 照相机📷 | 随时喊"照相机"，动的人喝（保留手中） |
| 6 | 摸鼻子👃 | 随时摸鼻子，最后摸的人喝（保留手中） |
| 7 | 逢7过7️⃣ | 报数逢7拍手跳过 |
| 8 | 厕所牌🚽 | 拥有才能上厕所（跨轮保留，可转让） |
| 9 | 自己喝🍺 | 抽到自己喝 |
| 10 | 神经病🤪 | 所有人不能和你对话（保留手中） |
| J | 上家喝⬆️ | 上一个抽牌的人喝 |
| Q | 下家喝⬇️ | 下一个抽牌的人喝 |
| K | 定K规则👑 | 定义下一个K的惩罚，4K结束本轮 |

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

然后访问 `http://localhost:3000`

### 生产环境部署

```bash
# 使用 PM2
npm install -g pm2
pm2 start ecosystem.config.js
```

## 🛠️ 技术栈

- **后端**: Node.js + Express + Socket.IO
- **前端**: HTML5 + CSS3 + JavaScript
- **实时通信**: WebSocket
